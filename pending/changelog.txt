🔧 **Claude Code アップデート**
**v2.1.113**
・CLIがバンドルされたJavaScriptの代わりに、ネイティブのClaude Codeバイナリ（プラットフォームごとのオプション依存関係を経由）を起動するよう変更
・`sandbox.network.deniedDomains` 設定を追加: `allowedDomains` ワイルドカードが許可する場合でも特定ドメインをブロック可能に
・フルスクリーンモード: 選択範囲を表示域の端まで広げる際にShift+↑/↓でビューポートがスクロールされるようになった
・マルチライン入力で、`Ctrl+A` と `Ctrl+E` が現在の論理行の先頭/末尾に移動するようになった（readlineの挙動に合わせた）
・Windows: `Ctrl+Backspace` で直前の単語を削除できるようになった
・レスポンスやBash出力中の長いURLが、複数行にわたってラップしてもクリック可能なままになった（OSC 8ハイパーリンク対応端末）
・`/loop` を改善: Escで保留中のウェイクアップをキャンセルでき、ウェイクアップは「Claude resuming /loop wakeup」と表示されるようになった
・`/extra-usage` がRemote Control（モバイル/Web）クライアントから利用可能になった
・Remote Controlクライアントが `@` ファイルオートコンプリート候補を照会できるようになった
・`/ultrareview` を改善: 並列チェックによる高速起動、起動ダイアログへのdiffstat表示、アニメーション起動状態を追加
・ストリーム途中で停止したサブエージェントが、サイレントハングの代わりに10分後に明確なエラーで失敗するようになった
・Bashツール: 最初の行がコメントのマルチラインコマンドでトランスクリプトにコマンド全体が表示されるようになり、UIスプーフィングの脆弱性を修正
・`cd` がno-opの場合、`cd <current-directory> && git …` の実行で権限プロンプトが出なくなった
・セキュリティ: macOSで `Bash(rm:*)` 許可ルール下での `/private/{etc,var,tmp,home}` パスが危険な削除対象として扱われるようになった
・セキュリティ: Bash拒否ルールが `env`/`sudo`/`watch`/`ionice`/`setsid` などのexecラッパーでラップされたコマンドにも一致するようになった
・セキュリティ: `Bash(find:*)` 許可ルールが `find -exec`/`-delete` を自動承認しなくなった
・MCP並行コールタイムアウト処理を修正: あるツールコールのメッセージが別のコールのウォッチドッグをサイレントに無効化する問題
・Cmd-backspace / `Ctrl+U` でカーソルから行頭まで削除できるよう修正
・セル内にパイプ文字を含むインラインコードスパンがある場合にMarkdownテーブルが壊れる問題を修正
・未送信テキストを入力中にセッションリキャップが自動実行される問題を修正
・`/copy` の「Full response」がGitHub・Notion・Slack貼り付け用にMarkdownテーブル列を整列しない問題を修正
・実行中のサブエージェントを表示中に入力したメッセージがそのトランスクリプトから非表示になり、親AIに誤って帰属される問題を修正
・Bash の `dangerouslyDisableSandbox` が権限プロンプトなしにサンドボックス外でコマンドを実行する問題を修正
・`/effort auto` の確認メッセージを修正: ステータスバーのラベルに合わせて「Effort level set to max」と表示するようになった
・「copied N chars」トーストが絵文字などのマルチコードユニット文字を過剰カウントする問題を修正
・`/insights` がWindowsで `EBUSY` でクラッシュする問題を修正
・終了確認ダイアログがワンショットスケジュールタスクを繰り返しと誤ってラベル付けする問題を修正: カウントダウンを表示するようになった
・フルスクリーンモードでスラッシュ/@ 補完メニューがプロンプト境界に密着しない問題を修正
・`CLAUDE_CODE_EXTRA_BODY` の `output_config.effort` がeffort非対応モデルへのサブエージェントコールおよびVertex AIで400エラーを引き起こす問題を修正
・`NO_COLOR` が設定されている場合にプロンプトカーソルが消える問題を修正
・`ToolSearch` のランキングを修正: 貼り付けたMCPツール名が説明一致の関連ツールではなく実際のツールを表示するようになった
・再開した長コンテキストセッションのコンパクト化が「Extra usage is required for long context requests」で失敗する問題を修正
・`plugin install` が既存プラグインと依存関係バージョンが競合する際に成功していた問題を修正: `range-conflict` を報告するようになった
・「Refine with Ultraplan」がトランスクリプトにリモートセッションURLを表示しない問題を修正
・処理に失敗したSDK画像コンテンツブロックがセッションをクラッシュさせる問題を修正: テキストプレースホルダーにフォールバック
・Remote ControlセッションがサブエージェントトランスクリプトをストリームしないFixed問題を修正
・Claude Code終了時にRemote Controlセッションがアーカイブされない問題を修正
・Bedrock Application Inference Profile ARN経由でOpus 4.7使用時に `thinking.type.enabled is not supported` 400エラーが発生する問題を修正
